   - task_count_total

   - pipeline_duration_seconds
   - gpu_memory_usage_bytes
   - queue_depth

   - retry_count

   - request_id
   - artifact_id
   - task_id
   - api_key_id
   - timestamp
   - level
   - message

4. Log levels:
   DEBUG
   INFO
   WARNING
   ERROR
   CRITICAL
