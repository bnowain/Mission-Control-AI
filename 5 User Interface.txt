PART 5: USER INTERFACE — MISSION CONTROL COCKPIT
Adaptive AI Execution OS
API-FIRST • OBSERVABLE • EXTENSIBLE • PROFESSIONAL

You are a senior frontend + full-stack architect.

Your task is to design and implement a comprehensive, API-first Mission Control UI for the Adaptive AI Execution Framework.

This UI must:

- Be modern, clean, and fast
- Be easy to navigate
- Be scalable
- Be modular
- Be fully API-driven
- Be production-ready
- Support expansion into distributed workers
- Support SQL access
- Support external API integrations
- Support reports and analytics
- Be usable by power users

This is not a chatbot UI.

This is an execution control cockpit.

------------------------------------------------------------
I. TECH STACK REQUIREMENTS
------------------------------------------------------------

Frontend:
- React + TypeScript
- Vite or Next.js (app router)
- Zustand or Redux Toolkit for state
- React Flow for DAG
- Monaco Editor for diff + SQL
- TailwindCSS + component library (shadcn/ui recommended)
- ECharts or Chart.js for analytics
- WebSocket support for live updates

Backend:
- Must use existing FastAPI API
- All calls must go through API layer
- No direct database access from frontend
